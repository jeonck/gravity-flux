# 🌀 GRAVITY FLUX — 중력 시프트 테트리스

> **중력을 지배하는 자가 보드를 지배한다.**
> 전통 테트리스의 규칙을 깨고, 보드의 중력을 90° 회전시켜 연쇄 클리어를 만드는 차세대 웹 테트리스.

## 🎮 핵심 메커니즘

| 메커니즘 | 설명 |
|---|---|
| **⚡ 중력 시프트** | `Q`/`E`로 쌓인 스택 전체를 90° 회전. 블록이 테트로미노 강체 단위로 재낙하하며 지형이 완전히 재편된다 |
| **🔗 체인 연쇄** | 클리어로 지지대가 사라지면 위 블록이 떨어져 새 라인을 완성 — 체인마다 점수 ×2 (최대 ×8) |
| **🔋 시프트 차지** | 라인 3개 클리어마다 차지 +1 (최대 3). 무한 남용 불가 — 발동 타이밍이 곧 전략 |
| **🔥 피버 모드** | 콤보·체인으로 게이지를 채우면 10초간 모든 점수 ×2 |
| **12×12 정사각 보드** | 스택 회전이 수학적으로 완전하게 성립하는 이 게임만의 보드 규격 |

## 🕹️ 조작

- **← →** 이동 · **↓** 소프트 드롭 · **Space** 하드 드롭
- **↑ / X** 회전 ↻ · **Z** 회전 ↺ · **C** 홀드 · **P** 일시정지
- **Q / E** 중력 시프트 (핵심!)
- 모바일: 화면 하단 터치 버튼 지원

## 🧠 전략

낮고 평평한 스택을 회전시키면 좁고 높은 기둥이 된다 — 손해.
**높게 쌓인 위기의 스택을 회전시키면 넓고 낮은 슬래브가 되며 라인이 폭발한다** — 역전.
위기가 곧 기회인 리스크 관리 게임.

## 🛠️ 기술

- Vanilla JS + Canvas API (프레임워크 0, 의존성 0)
- 테트로미노 강체 캐스케이드 물리 (연결 컴포넌트 기반 그룹 낙하)
- WebAudio 신디사이저 사운드 (에셋 0)
- 7-bag 랜덤, 고스트 피스, 홀드, 월킥 회전

## 🚀 실행

```bash
python3 -m http.server 8642   # 또는 아무 정적 서버
```

## 🤖 개발 과정

게임 디자이너 · 풀스택 개발자 · UX 기획자 · 비평가 4개 에이전트의 토론으로 컨셉을 선정하고,
루프 엔지니어링(구현 → 브라우저 자동 검증 → 결함 수정 반복)으로 완성.
비평가 에이전트가 "열 압축 캐스케이드는 체인이 수학적으로 불가능하다"는 결함을 잡아내
강체 그룹 낙하 물리로 재설계된 것이 핵심 개선점.
